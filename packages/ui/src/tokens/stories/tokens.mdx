# Nexus Design Tokens

A comprehensive catalog of all design tokens in the Nexus design system. These tokens form the foundation of our UI components with a modern color palette featuring violet brand colors and slate neutrals.

---

## Colors

The Nexus color system is built on a two-tier architecture: **Primitive colors** provide the raw palette, while **Semantic colors** map these primitives to specific use cases.

Foundational color scales that form the basis of the design system.

## Usage

Import tokens in your components:

```typescript
import { colors, gradients, spacing, shadows } from '@repo/ui/tokens';
import styled from 'styled-components';

export const Button = styled.button`
  background-color: ${colors.semantic.brand.primary};
  padding: ${spacing[4]};
  box-shadow: ${shadows.md};
  background: ${gradients.brand.accent};
`;
```

For themed values, use the theme prop:

```typescript
import styled from 'styled-components';

export const Card = styled.div`
  background-color: ${({ theme }) => theme.colors.semantic.background.base};
  border-color: ${({ theme }) => theme.colors.semantic.border.default};
`;
```
